# 本地生活服务 AI 原生原型 - 产品需求文档 (PRD)

## 1. 项目背景

针对抖音等内容平台"种草-决策-到店"的闭环，利用 LLM 能力解决商家营销成本高、达人创作效率低、用户搜索链路长的问题。

## 2. 核心用户路径

**流量端：** 短视频内容驱动 → 搜索意图触发

**转化端：** POI（门店）详情页 → 优惠券/预约 → 到店核销

**AI 赋能点：** 极速产出高质量营销素材，缩短用户从"看到"到"下单"的思考时长。

## 3. 功能模块设计

### Module A: AI 探店脚本专家

- **输入：** 门店类型（如火锅）、特色关键词、目标人群
- **输出：** 包含"黄金 3 秒开头 + 痛点挖掘 + 反转点 + 引导下单（CTA）"的短视频脚本

### Module B: 商家营销文案引擎

- **输入：** 促销活动、门店氛围图描述
- **输出：** 适配抖音/小红书风格的标题与正文，包含热门话题标签

### Module C: 城市美食推荐助手（LBS 模拟）

- **输入：** 用户当前位置、口味偏好、时段
- **输出：** 语义化推荐列表（模拟调用高德/抖音 POI 接口）

## 4. 成功指标 (MVP 阶段)

- 生成内容在逻辑上符合抖音推流算法（高停留时长、高互动）
- 原型可运行，交互流程顺畅
